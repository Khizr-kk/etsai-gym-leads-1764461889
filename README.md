# GymLeads India

Auto-generated by ETS-AI 10x.

## Idea Summary
A SaaS platform designed for small local gyms in India to manage leads from WhatsApp and Instagram.

## Problem Statement
Small local gyms in India struggle to effectively capture, track, and convert leads generated through WhatsApp and Instagram into paying members.

## Target Persona
Indian small gym owner

## MVP Features
- WhatsApp Business API integration for lead capture and messaging
- Instagram Graph API integration for lead capture and messaging
- Unified Lead Inbox/Dashboard to view all incoming leads from WhatsApp and Instagram
- Detailed Lead Profile (contact info, source, interaction history, status)
- Ability to manually update lead status (New, Contacted, Follow-up, Converted, Lost)
- Basic messaging functionality to reply to leads directly from the platform
- ML-driven Lead Scoring (High, Medium, Low conversion potential)
- Basic reporting on lead status distribution and conversion rates
- User authentication and profile management for gym owners

## Tech Stack
Python (Django/FastAPI) for backend (good for ML integration), React/Vue.js for a responsive frontend, PostgreSQL for the database, and AWS/Google Cloud for scalable infrastructure. WhatsApp Business API and Instagram Graph API for integrations.
